Objective

Create a website that is visually appealing with great information to help recruiters or collaborators learn more about me and the type of projects I have completed.

Project Difficulties

The beginning stages of this website was very simple due to a very bad responsive page system. Had difficulties with creating customized layouts for each pages and it was very time consuming to find the right values for the layouts.

My Solution

Decided to use Bootstrap framework put their own classes and user interfaces to good use. After countless hours of learning and testing the framework, I was able to use almost every class to my website and succesfully created a more responsive page system to different screen ratios and on mobile as well. It also added flexibility to my Angular routing and css animations.

Features
  • Hierarchy Routing Animations
  • Interactive images
  • Visually appealing information
  • Uses Netlify page hosting for continuos deployment
Features to Add or Fix
  • Dynamic routing for the different project pages.
  • Add .NET Core support to involve back-end development such as adding SMTP protocol for a contact page or to simply save information in a database.
  • Make the page a lot more mobile friendly by adding/removing elements based on the size of the screen.
