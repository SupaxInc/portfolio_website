Project Difficulties

My partner and I had to get through a certain problem that displayed multiple categories of obstacles such as lack of time, resources, money, etc. We had to save the trained brain of the neural network from our simulation software and send it to the remote controlled car.

Our Solution

Since we could not afford to buy our own cloud server or wait for our school's database admin to create us our own server for our project (would have delayed us two weeks of work).

We decided to look at the thousands of python libraries. We were able to find a library that was able to actually connect to the google drive cloud and we were able to read and write to a text file.

Features

  • Able to manually control the car or set it on autopilot with android mobile app.
  • Actual car will avoid obstacles and is able to learn its path more efficiently.
  • Fully customizable settings for the AI and Car.
  • Singleton design pattern used to control spefic areas of the simulation software.
